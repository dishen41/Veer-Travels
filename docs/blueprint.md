# **App Name**: Veer Travels - Customized Group Trip Planner

## Core Features:

- Destination Showcase: Displays destinations in an interactive grid (Domestic and International).
- Trip Inspiration Gallery: Showcase group trip themes (Corporate, Student, etc.) with image galleries and testimonials.
- Quote Request: A contact form allowing users to submit their custom travel requirements.
- Detailed Destination Views: Displays detailed information, images, and top attractions for each destination upon selection.
- Mobile-Responsive Design: Ensures the application is fully responsive and looks perfect on all devices (desktop, tablet, mobile).
- AI-Powered Travel Recommendations: Suggest destinations and itinerary options based on user-provided preferences. The tool leverages historical travel data and trending destinations to personalize recommendations.

## Style Guidelines:

- Primary color: Ocean blue (#3498db) for a calming, professional feel.
- Background color: Light blue (#eaf2ff), creating a clean and airy atmosphere.
- Accent color: Coral (#FF7F50) to highlight key CTAs and elements.
- Headline font: 'Poppins' (sans-serif) for a contemporary, precise feel.
- Body font: 'PT Sans' (sans-serif) to ensure readability and consistency.
- lucide-react icons (scalable vector icons) to be used consistently throughout the site to visually represent destinations, services, and features.
- Clean and minimal layout to ensure focus on visuals and ease of navigation. Mobile-first approach to guarantee responsiveness on all devices.
- Subtle animations and transitions for interactive elements (e.g., hover effects on destinations) to improve user engagement without being distracting.
